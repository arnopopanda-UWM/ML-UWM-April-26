## Git command for noticing all (.)  changes that are made.

git add .

## specific file upload

git add CC_Fraud_detection.ipynb

## saves a snapshot of your changes with a message.

git commit -m "Describe the change"

## Push to GitHub: uploads your commit to GitHub.

git push origin main

## Showing shich files are modified/deleted

git status

## Unstage specific files

git restore --staged RandomForest_Alternative.py

## Unstage ALL staged files

git reset
